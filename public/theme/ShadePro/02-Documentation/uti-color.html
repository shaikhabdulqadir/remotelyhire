Color utilities

shadePro includes a wide range of colors utility classes to modify an elementâ€™s text, icon, background and border color scheme appearance.

Color variables & values

Any color key listed in the _theme-color-maping.scss using the variables declared in _theme-color-control.scss files' will automaically generate grayic's utility classes for text color, background color, border color and opacity color.

Text Color Class Notation

Text color related utility classes are named using the format text-{colorName},

Where colorName is one of the key from the _theme-color-maping.scss's '$theme-colors' array.

.text-green

.text-blackish-blue

BG (Background) Color Class Notation

Background color related utility classes are named using the format bg-{colorName},

Where colorName is one of the key from the _theme-color-maping.scss's '$theme-colors' array.

.bg-green

.bg-blackish-blue

Opacity Text Color Class Notation

Opacity Text color related utility classes are named using the format gr-color-{colorName}-opacity-{opacityValue},

Where colorName is one of the key from the _theme-color-maping.scss's '$theme-colors' array.

and opacityValue is a number ranging from 0 to 9 creating 10 variation of opacity from transparent to consistently darkening solid color.

.gr-color-green-opacity-3

.gr-color-blackish-blue-opacity-7

Opacity BG (Background) Color Class Notation

Background Opacity color related utility classes are named using the format gr-bg-{colorName}-opacity-{opacityValue},

Where colorName is one of the key from the _theme-color-maping.scss's '$theme-colors' array.

and opacityValue is a number ranging from 0 to 9 creating 10 variation of opacity from transparent to consistently darkening solid color.

.gr-bg-green-opacity-3

.gr-bg-blackish-blue-opacity-7

Border Color Class Notation

Background color related utility classes are named using the format border-{colorName},

Where colorName is one of the key from the _theme-color-maping.scss's '$theme-colors' array.

.border.border-green

.border.border-blackish-blue

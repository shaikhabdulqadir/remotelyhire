Typography

shadePro includes a wide range of shorthand responsive Typography of various font sizes utility classes to modify an elementâ€™s text appearance.

How It works

Assign responsive-friendly font-size values to elements with shorthand classes. Classes are built using bootstrap's convention and also using variable values ranging from .5rem to 5rem.

Notation

Text Font utilities apply to all breakpoints, from xs to xl, have no breakpoint abbreviation in them. This is because those classes are applied from min-width: 0 and up, and thus are not bound by a media query and they act as responsive-friendly on their own. This behavior is controlled by a single variable boolean value named $enable-responsive-font-sizes which is set to true in our themes by default. You can change it to false if you don't want the automatic responsive behavior and want responsive control on your hand. This variable can be found on app/scss/controls/_theme-font-control.scss .

The classes are named using the format gr-text-{sizeNumber}.

Where sizeNumber is one of:

  • 1 - (by default) for classes that set the font-size to $font-size-base * 5

  • 2 - (by default) for classes that set the font-size to $font-size-base * 3.75

  • 3 - (by default) for classes that set the font-size to $font-size-base * 3

  • 4 - (by default) for classes that set the font-size to $font-size-base * 2.25

  • 5 - (by default) for classes that set the font-size to $font-size-base * 2

  • 6 - (by default) for classes that set the font-size to $font-size-base * 1.5

  • 7 - (by default) for classes that set the font-size to $font-size-base * 1.3125

  • 8 - (by default) for classes that set the font-size to $font-size-base * 1.1875

  • 9 - (by default) for classes that set the font-size to $font-size-base * 1.0625

  • 10 - (by default) for classes that set the font-size to $font-size-base

  • 11 - (by default) for classes that set the font-size to $font-size-base * 0.9375

  • 12 - (by default) for classes that set the font-size to $font-size-base * 0.8125

  • 13 - (by default) for classes that set the font-size to $font-size-base * 0.625

  • 14 - (by default) for classes that set the font-size to $font-size-base * 0.5

    • N.B: $font-size-base is equivalent of your browser setting's default font size, 16 px or 1 rem in general.

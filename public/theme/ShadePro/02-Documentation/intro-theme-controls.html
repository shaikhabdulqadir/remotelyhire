Introduction to Theme Controls

Theme controls are what makes your theme more controllable and a lot easier to customize without changing css or custom coding. Let's dive deeper!

Control Blocks and Utility Classes:

For easier accessibility, the theme is sorted into basic theme blocks like typography, colors, variables, spacings, components, etc.

We have seperated each block or controls to their corresponding scss partial files.

The variable control folder, controls is situated in app/scss/ and the partials contained in this folder with their roles are listed below:

  • _theme-variables.scss: imports all variables, and variable containing partials in this file.
  • _theme-color-control.scss: contains all color related variables used throughout the theme.
  • _theme-color-maping.scss: contains color maping arrays consisting of the _theme-color-control.scss's variables based on which the theme's background, text and border colors' utility classes are generated.
  • _theme-font-control.scss: contains all font related variables used to generate texts' utility classes such as font sizes, font families, font's responsive behavior controller variables, heading's and body text's color variables etc.
  • _theme-form-control.scss: contains all form's element related variables such as input's sizes, spacings, border-radius, button related variables etc.
  • _theme-grid-control.scss: contains bootstrap's unique grid breakpoint holding array "$grid-breakpoints". Adding any custom breakpoint key-value properties in this array will generate custom grid utility classes like bootstrap's automatically.
  • _theme-spacing-control.scss: contains spacing value holding array "$gr-spacers". Adding any custom spacing key-value properties in this array will generate custom spacing utility classes like bootstrap's automatically, but with 'gr-' prefix, e.g: 'gr-mb-5' is equavalent of 1 rem * 1.25 (assuming browser default as 1rem or 16px).
  • _theme-media-query.scss: contains custom media query mixins used for easier development purpose.
  • _theme-mixins.scss: contains mixins used for generating classes which will be used in resetting text or headings' line-height (the extra spacing above and below the baseline).

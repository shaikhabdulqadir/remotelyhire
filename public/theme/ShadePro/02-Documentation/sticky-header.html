Activating Sticky Header

You can easily enable/disable sticky Header on shadePro

For activating sticky header first you need to find the .site-header markup:



<!-- Site Header-->
<div class="site-header">
  ...
</div>

Then Place the .sticky-header class after the .site-header class



<!--  Site Header -->
<div class="site-header sticky-header">
  ...
</div>

By default it will be sticky on desktop only.
For enabling sticky header on mobile also you have to add .mobile-sticky-enable class along with the .sticky-header class

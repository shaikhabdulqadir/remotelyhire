Edit the Color Scheme

You can easily transform the color scheme of your site using SCSS.

Editing the default color scheme variables

Editing the default color scheme variables will change the colors in main.css and result the theme colors to change. Follow below instructions for changing heading titles, paragraph texts, buttons and theme base colors.

  1. Open the _theme-color-control.scss file

    The _theme-color-control.scss file is located in the app/scss/controls directory of the theme package.

  2. Change the variable values to your preferred colors

    Inside _theme-color-control.scss, Modify the HEX values to change the color scheme. You can also define and add new color variables here.

    //--> All Variable Colors
  $white:    #fff;
  $ghost:    #fdfdff;
  $smoke:    #f8f8f8;

  $gray-110: #fcfdfe;
  $gray-120: #f4f7fa;
  $gray-210: #e7e9ed;
  $gray-opacity:   rgba(231, 233, 237, 0.13);

  $storm:    #7d818d;


  $red:                   #f64b4b;
  $blue:                  #473bf0;
  $sky-blue:              #1082e9;
  $green:                 #68d585;
  $green-shamrock:        #2bd67b;
  $blackish-blue:         #161c2d;
  $mirage:                #131829;
  $yellow:                #f7e36d;

//--> Setting up the primary color
$primary:       $blue;
$secondary:     $green;

// Based on th

$headings-color:            var(--color-headings); 
$text-color:                var(--color-text);
  3. Generate color classes(i.e, text-blue, bg-blue, border-blue) for your new color variables by adding the new variables in the $theme-colors array inside the _theme-color-maping.scss file. 
    //-->Currently All Available Colors
  
  "primary":    $primary,
  "secondary":  $secondary,
  "red":              $red,
  "green":            $green,
  "green-shamrock":   $green-shamrock,
  "blue":             $blue,
  "sky-blue":         $sky-blue,
  "yellow":           $yellow,
  "blackish-blue":    $blackish-blue,
  "black":            $black,
  "mirage":           $mirage,
  "white":            $white,
  "smoke":            $smoke,
  "storm":            $storm,
  "ghost":            $ghost,
  "gray-1":           $gray-110,
  "gray-2":           $gray-120,
  "gray-3":           $gray-210,
  "gray-opacity":     $gray-opacity,

    To add your new color variable in the $theme-colors array, Add your color and variable name right below "gray-opacity": $gray-opacity, and follow the example given below:

    
  "yourColorName":    $yourColorVariableName,

    Make sure $yourColorVariableName and "yourColorName" is same.

NOTE: In the purpose of controlling themes dark and light version colors, such as title, paragraph, border and background colors, Some SCSS variables hold css variables as values in some cases. If you don't Need The light/Dark Mode, you can change the css variable values into hex/Rgba/HSL color code as value of the scss variable.

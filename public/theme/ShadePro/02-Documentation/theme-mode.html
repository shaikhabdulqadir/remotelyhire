Theme Mode Guide

shadePro comes with the attractive feature of "Dark/Light" mode

Activating the Theme Mode

Editing the default Theme Mode will change the theme version light to dark or dark to light. Follow instructions below for changing the default Theme Mode

  1. Open the _theme-mode-activation.scss file

    The _theme-mode-activation.scss file is located in the app/scss/theme-mode-control directory of the theme package.

  2. Change the variable values to your preferred mode

    Inside _theme-mode-activation.scss, change the $theme-mode-active SCSS variable value to "light" or "dark".By default the light version is activated. 

    $theme-mode-active: 'light';

Activating the Theme Mode Switcher Panel

Theme Mode switcher Panel enables you switch the theme mode anytime in live, follow the steps to activate the theme mode switcher panel :

  1. Step 1:

    Link the theme mode script and Styles to your page. the script and style is situated in "app/plugins/theme-mode-switcher"

    <link rel="stylesheet" href="./plugins/theme-mode-switcher/switcher-panel.css">
<script src="./plugins/theme-mode-switcher/gr-theme-mode-switcher.js"></script>
  2. Step 2:

    Add the following attributes to the body tag to show Theme switch panel:

    <body data-theme-mode-panel-active>

By default the light version is activated.You can change the default activation of theme mode by adding and changing the data-theme attribute value. ex: data-theme="dark"
